# Sync Folder Accessibility Statement

Last updated: June 18, 2026

Ofir Meguri aims to make Sync Folder usable by people with disabilities and to
improve accessibility as issues are identified.

## Accessibility measures

The app is designed to support:

- Android TalkBack and standard screen-reader navigation.
- System font scaling and display scaling.
- Clear text labels for interactive controls.
- Keyboard and switch-access behavior provided by standard Android controls.
- Status and error announcements through accessible live regions.
- Touch targets and color contrast based on standard Material components.

## Known limitations

Accessibility testing is ongoing, and the app has not been certified by an
independent accessibility assessor. Some behavior can depend on the Android
version, device manufacturer, system file picker, Google Sign-In, or Google
Drive interfaces that are outside the app's control.
